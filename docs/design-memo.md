# DevTycoon 설계 메모 (MVP 원본)

> 최초 MVP 프로토타입(`prototype/devtycoon.html`)과 함께 작성된 원본 설계 메모.

- **자원:** 코드(LoC) / 수익(₩) / 유저 / 기술부채 / 경력
- **루프:** 코드 짜기 → 기능 출시(유저↑) → 채용(자동생산) → 부채 관리 → 환생
- **핵심 긴장:** 부채가 코드·수익 둘 다 깎음 (`prodMult = 1/(1+debt/120)`)
- **환생:** 누적수익 → 경력, 경력당 전역 +2%
- **튜닝 상수:** `<script>` 상단 상수 + `DEVS` 배열에 집중

## MVP 기준 다음 작업 (원본 메모)
